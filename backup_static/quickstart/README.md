# Quickstart Guide

Get up and running with OpenRockets in minutes. We handle the infrastructure so you can focus on making an impact.

## Step 1: Register your Organization

Before you can utilize our infrastructure, your organization must be registered with ZeroProfit™ or be an officially recognized youth-led initiative.

```text
1. Visit zeroprofit.org
2. Submit your organization's details
3. Wait for community approval (typically 24-48 hours)
```

## Step 2: Join the Network

Once approved, you'll receive access credentials to our central dashboard where you can provision services like hosting, domains, and collaboration tools.

### Initial Configuration

Ensure you have your API keys generated from the dashboard. Set them up securely:

```bash
export OPENROCKETS_API_KEY="sk_test_..."
```

For more details on how our systems connect, view the [Architecture Overview](../architecture/index.html).
