# Welcome to OpenRockets Docs

Build and integrate with the infrastructure designed for exceptional youth-led nonprofits. Start connecting your services with rich presence, seamless collaboration, and more.

## Overview

OpenRockets provides robust, secure, and scalable infrastructure to support teenagers building solutions that can change the world. This documentation will guide you through getting started, establishing partnerships, and setting up the backstage services you need to thrive.

## Features

- **Global Network**: Connect your youth-led project to a global network spanning the Americas, Europa, Asia-Pacific, Oceania, and beyond.
- **Enterprise Infrastructure**: High quality security, partnerships, networks, and platforms provided completely free of charge.
- **Community First**: Everything we do is to empower minors and teenagers.

Read more in our [Quickstart Guide](../quickstart/index.html) to begin your journey.
