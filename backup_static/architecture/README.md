# Architecture Overview

Understand how OpenRockets scales and secures your deployments.

## Infrastructure Layer

We leverage industry-standard providers and abstract the complexity away so your team can focus on making an impact. Our layer handles load balancing, DDoS protection, and global edge caching automatically.

## Security by Default

Security is our top priority. We provide end-to-end encryption for all internal communication and secure SSO for your organization's members.

### Core Components

1. **Edge Network**: Cloudflare-backed edge network ensuring lightning-fast response times globally.
2. **Compute**: Distributed containerized workloads.
3. **Database**: Managed replication and automatic backups.
